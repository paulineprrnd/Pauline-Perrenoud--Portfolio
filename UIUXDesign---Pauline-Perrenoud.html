UI UX design.

Sites web, applications et création de leur design

Why so serious ?

Ce projet scolaire est intitulé "Boost my movie". Le but est de faire la promotion dun film en codant un site à son propos, en présentant le synopsis, les personnages principaux et leurs acteurs, et des images. Nous avons codé nos sites en HTML-CSS en utilisant Javascript pour certaines animations. Nous avons beaucoup été aidés dans nos cours de code.

SJ'ai donc créé ce site de  A à Z. J'ai réalisé la maquette sur  Adobe Xd et le code HTML CSS sur Webstorm. C'était ma toute première expérience de code.
